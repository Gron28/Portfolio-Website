## Project Structure

```
portfolio-website/
├── src/                    # Source files
│   ├── partials/          # Reusable HTML components
│   ├── projects/          # Individual project pages
│   └── freelance/         # Freelance services page
├── css/                   # Stylesheets
├── js/                    # JavaScript files
├── images/               # Media assets
├── cv/                    # Resume files
├── projects/             # Built project pages
├── freelance/            # Built freelance page
├── index.html            # Main portfolio page
├── build.js              # Build script
└── README.md             # This file
```

## Build Process

The project uses a custom Node.js build script to assemble HTML partials:

```bash
node build.js
```

This combines navigation, footer, and content partials into complete HTML pages.
